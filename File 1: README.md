# Turkmine Gold Prospectivity Mapping

## Integrated Remote Sensing and GIS for Gold Mineralisation Mapping at Turkmine Open Pits, Zimbabwe

### Overview

This repository contains all Python scripts developed for the BSc Honours research project titled:

**"Enhanced Gold Mineralisation Mapping Using Integrated Remote Sensing and GIS Techniques at Turkmine Open Pits"**

The project integrates LiDAR, Sentinel-1 SAR, and PRISMA hyperspectral data to map gold mineralisation potential using a fuzzy weighted overlay approach.
